# 🍔 Food Court Management System

A console-based **Food Court Management System** developed using **C++** and **Object-Oriented Programming (OOP)** concepts.
This project helps manage food stalls, menu items, and customer interactions through different portals.

---

# 📌 Project Overview

The Food Court Management System is a menu-driven application designed to manage food stall records efficiently.
The system contains three major portals:

* **Admin Portal**
* **Food Stall Portal**
* **Customer Portal**

The project demonstrates the practical implementation of:

* Object-Oriented Programming
* File Handling
* Menu-Driven Programming
* Authentication System
* CRUD Operations
* Input Validation

---

# 🚀 Features

## 👨‍💼 Admin Portal

* Admin Login Authentication
* Add Food Stall
* Delete Food Stall
* View Food Stall Details
* Manage Food Stall Records

---

## 🍽️ Food Stall Portal

* Stall Owner Login
* Add Food Menu Items
* Delete Food Items
* Modify Food Stall Profile
* View Food Stall Menu
* Update Food Item Details

---

## 👥 Customer Portal

* View Available Food Stalls
* Search Food by Category
* View Food Stall Menus
* Search Food Items

# ⚙️ How to Run the Project

## Step 1: Clone Repository

```bash
git clone <repository-link>
```

---

## Step 2: Open Project Folder

```bash
cd FoodCourtManagement
```

---

## Step 3: Compile Project

```bash
g++ main.cpp src/*.cpp -o foodcourt
```

---

## Step 4: Run Application

### Linux / Mac

```bash
./foodcourt
```

### Windows

```bash
foodcourt.exe
```
